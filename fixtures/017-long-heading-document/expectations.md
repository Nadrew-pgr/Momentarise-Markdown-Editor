# Expectations

- Preserve all headings, heading levels, section order, repeated structure, and paragraphs.
- Normalized output may enforce a final newline but must not collapse sections.
- Opaque handling is not expected for standard headings and paragraphs.
- Source-only handling is not required, but folding sidecar state must not mutate Markdown source.
- Render as a navigable heading outline suitable for folding tests.
