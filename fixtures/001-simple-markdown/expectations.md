# Expectations

- Preserve the heading hierarchy, paragraphs, emphasis, strong text, inline code, link target, and list item order.
- Normalized output may standardize emphasis markers, but must not change visible meaning.
- Opaque handling is not expected for known Markdown in this fixture.
- Source-only handling is not required unless a future renderer cannot display a construct.
- Render as ordinary Markdown with headings, paragraph text, inline formatting, link, and bullet list.
